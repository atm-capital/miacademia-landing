# miacademia-landing

Landing page para [miacademia.app](https://miacademia.app) — bot WhatsApp para academias deportivas y de danza.

## Stack

- HTML estático + Tailwind CSS (CDN)
- Plus Jakarta Sans + Inter (Google Fonts)
- Deploy en Vercel (static)

## Estructura

- `index.html` — página única con todas las secciones
- `vercel.json` — config Vercel (cache headers, security)
- `demo.mp4` — video demo embebido
- `og-image.png` / `og-image-square.png` — social previews
- `favicon.png`, `logo-*.png` — assets de marca
